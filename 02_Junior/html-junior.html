Основы HTML

Итак, редактор выбран и настроен. Можно приступать непосредственно к обучению.

Как мы уже знаем из вводных уроков, верстка сайта выполняется с помощью трех основных технологий:

  • HTML - язык гипертекстовой разметки. Создает логическую структуру - каркас нашей верстки
  • CSS - каскадные таблицы стилей. Создает визуальную модель - описывает то, как наша верстка будет выглядеть.
  • JavaScript - язык программирования. Добавляет интерактивности к нашей верстки, т.е. взаимодействие с пользователем. Выполняет разного рода вычисления и может использоваться для обмена данными

Первым делом беремся за язык разметки гипертекста, он же HTML. НО, беремся не сразу на весь язык а только за необходимую его часть!

То есть открывать справочник и учить все подряд теги совсем не эффективно, более того вредно!

Из всех возможных HTML - тегов в работе, чаще всего, используется лишь небольшая их часть. Но даже эту часть я советую разделить на группы и изучать их в разный период своего развития!

Инсайт

Собственно, первым делом мы беремся за самые простые и понятные теги и кроме синтаксиса, вложенности и основной конструкции HTML-страницы, то есть тегов HTML BODY HEAD TITLE, обращаем внимание на следующие теги:

  • Так называемый строительный тег DIV.
  • Теги для заголовков H1-H6,
  • Теги для текста P, SPAN,
  • Конструкции списков UL-LI, OL-LI,
  • Тег для вывода картинок IMG
  • Тег для вывода ссылки A
  • Также берем на вооружение теги основной структуры страницы HEADER FOOTER и MAIN
  • Теги крупных блоков страницы NAV SECTION ARTICLE ASIDE

Работаем с ними и их основными атрибутами.

И как вы понимаете, это совсем немного информации, и под силу даже совсем начинающему.

Но, её вполне достаточно, чтобы написать свои первые HTML конструкции, познакомится с блочными, строчными и блочно-строчными тегами. А также с основами семантики.

Семантика

Выделение крупных смысловых блоков страницы с помощью главных структурных тегов:

  • HEADER
  • MAIN
  • FOOTER

Привыкаем выделять крупные смысловые разделы в блоках с помощью семантических тегов:

  • NAV
  • SECTION
  • ARTICLE
  • ASIDE

Уметь правильно применять теги

  • DIV - для стилизации блоков контента;
  • SPAN - для стилизации мелких групп текста.

Контентная модель HTML5

До появления HTML5 существовало две основные категории элементов — блочные и строчные. Такое разделение не совсем точно описывало различные типы содержимого, поэтому в спецификации HTML5 модель содержимого была расширена.

Контентная модель (content model), или модель содержимого, описывает, какой тип содержимого следует ожидать внутри элемента и какие элементы могут быть вложены в другие элементы.
