Создание списков

Списки встречаются практически везде. Список дел определяет что сделано. Навигационные маршруты предлагают пошаговый список направлений. Для рецептов требуется список ингредиентов и список инструкций. Списки предоставляют возможность упорядочить и систематизировать разные данные и представить их в наглядном и удобном для пользователя виде.

Когда мы хотим использовать список на сайте, HTML предлагает три разных типа на выбор:

  • маркированный (неупорядоченный) список
  • нумерованный (упорядоченный) список
  • список описаний

Списки, также, могут вкладываться один в один

  • Вложенные списки

Какой тип списка выбрать для использования и применять ли список вообще — зависит от содержимого и семантически наиболее подходящего варианта для его отображения.

Маркированные списки

<ul> <li>

"Живой стандарт"

Маркированный или неупорядоченный список определяется тем, что перед каждым элементом списка добавляется небольшой маркер, обычно в виде закрашенного кружка.

Сам список формируется с помощью элемента <ul> (от англ. unordered list — неупорядоченный список)

Каждый пункт списка должен начинаться с элемента <li> (от англ. List Item — пункт списка). Если к <ul> применяется таблица стилей, то элементы <li> наследуют эти свойства.

С элементом <ul> связаны следующие особенности:

  • в том месте, где встречается <ul>, браузер автоматически добавляет перенос строки;
  • у списка имеются отступы сверху и снизу;
  • маркеры по умолчанию отображаются в виде закрашенного кружка;
  • каждый элемент списка сдвигается вправо по отношению к основному тексту.

Синтаксис

                        <ul>
                            <li>пункт маркированного списка</li>
                        </ul>
Внутри <li> не обязательно размещать только текст, это могут быть и другие элементы, вроде абзацев <p>.

Закрывающий тег

Обязателен

Атрибуты

Для этого элемента доступны универсальные атрибуты

class

Определяет имя класса, которое позволяет связать элемент со стилевым оформлением.

id

Указывает имя стилевого идентификатора.

Нумерованные списки

<ol> <li>

"Живой стандарт"

Нумерованные или упорядоченные списки представляют собой набор пунктов с их порядковыми номерами.

Устанавливает нумерованный список элемент <ol> (от англ. ordered list — упорядоченный список)

Каждый пункт списка должен начинаться с элемента <li> (от англ. List Item — пункт списка). Если к <ol> применяется таблица стилей, то элементы <li> наследуют эти свойства.

  1. в том месте, где встречается <ul>, браузер автоматически добавляет перенос строки;
  2. у списка имеются отступы сверху и снизу;
  3. по умолчанию применяется список с арабскими числами (1, 2, 3,…),;
  4. каждый элемент списка сдвигается вправо по отношению к основному тексту.

Синтаксис

                            <ol>
                                <li>Первый пункт</li>
                                <li>Второй пункт</li>
                                <li>Третий пункт</li>
                            </ol>
Внутри <li> не обязательно размещать только текст, это могут быть и другие элементы, вроде абзацев <p>.

Закрывающий тег

Обязателен

Атрибуты

type
Устанавливает вид маркера списка.
reversed
Нумерация в списке становится по убыванию (3,2,1).
start
Задаёт число, с которого будет начинаться нумерованный список.
value
Число, с которого будет начинаться нумерованный список. Применяется только для элемента <li>

Также для этого элемента доступны универсальные атрибуты class и id

type

Устанавливает вид нумерации в упорядоченном (нумерованном) списке.

Синтаксис

<ol type="A | a | I | i | 1">...</ol>

Значения

В качестве маркеров могут быть следующие значения:

  • заглавные латинские буквы =" A "
  • строчные латинские буквы =" a "
  • заглавные римские цифры =" I "
  • строчные римские цифры =" i "
  • арабские цифры =" 1"
<ol type="A">...</ol>
  1. Asus
  2. Lenovo
  3. Samsung
<ol type="a">...</ol>
  1. Asus
  2. Lenovo
  3. Samsung
<ol type="I">...</ol>
  1. Asus
  2. Lenovo
  3. Samsung
<ol type="i">...</ol>
  1. Asus
  2. Lenovo
  3. Samsung
<ol type="1">...</ol>
  1. Asus
  2. Lenovo
  3. Samsung

Значение по умолчанию

арабские цифры =" 1"

reversed

Меняет нумерацию в списке на обратный порядок, вместо 1,2,3 будет выводиться 3,2,1.

Синтаксис

<ol reversed>...</ol>

Значения

Нет

start

Устанавливает номер, с которого будет начинаться нумерованный список. При этом не имеет значения, какой тип списка установлен с помощью type, атрибут start одинаково работает и с римскими и с арабскими числами.

Синтаксис 

                        <ol start="число">
                            <li>пункт  списка</li>
                        </ol>

Значения

Нет

value

Атрибут устанавливает номер, с которого будет начинаться список. Применяется только для элемента <li> нумерованных списков. При этом не имеет значения, какой тип списка установлен с помощью type, атрибут start одинаково работает и с римскими и с арабскими числами.

Синтаксис 

                        <li value="число">...</li>

Значения

В качестве значения указывается любое целое положительное число. При этом неважно, какой тип нумерации установлен, даже если в качестве списка используются латинские буквы. Если одновременно для списка применяются атрибуты start и value, то последний имеет большее преимущество, и нумерация отображается с числа, указанного value,

Списки описания

<dl> <dt> <dd>

"Живой стандарт"

Такие списки применяются для обозначения нескольких терминов и их описаний, как в глоссарии, к примеру. Список описаний состоит из двух элементов — термина и его описания.

Создание списка описания в HTML происходит с помощью элемента <dl> (от англ. description list — список описаний)

Термин - элементом <dt> (от англ. description list term — термин списка описаний)

Описание термина — с помощью элемента <dd> (от англ. description list description — описание списка описаний)

Синтаксис

                        <dl>
                            <dt> Термин 1 </dt>
                                <dd> Описание термина 1 <dd> 
                            <dt> Термин 2 </dt>
                                <dd> Описание термина 2 <dd>
                        </dl>

Вложенные списки

Вложенные или многоуровневые списки предназначены для организации сложной иерархической структуры текста, обычно таких документов, как юридические или технические. Реально на веб-странице нельзя автоматически ввести многоуровневую нумерацию типа использования подпунктов 1.1 или 2.1.3. Поэтому приходится вводить числа отдельно или упрощать отображение списка.

  1. Украинская кухня
    • Борщ
    • Холодец
    • Галушки
  2. Кавказкая кухня
  3. Итальянская кухня

Синтаксис

                        <ol>
                            <li>Украинская кухня
                                <ul>
                                    <li>Борщ</li>
                                    <li>Холодец</li>
                                    <li>Галушки</li>
                                </ul>
                            </li>
                            <li>Кавказкая кухня</li>
                            <li>Кавказкая кухня</li>
                        </ol>

Поскольку вложенные списки могут немного запутать и отобразить нежелательные стили, если они сделаны неправильно — давайте быстро их просмотрим. Элементы <ul> и <ol> могут содержать только элементы <li>. Элемент <li> может содержать любой обычный элемент, какой пожелаете. Однако элемент <li> должен быть прямым потомком либо элемента <ul>, либо <ol>.

Стоит также отметить, что когда списки вложены внутрь других списков, их маркеры будут меняться в зависимости от глубины вложения. В предыдущем примере маркированный список, вложенный в нумерованный, использует в качестве маркеров кружок вместо точки. Это изменение происходит потому, что маркированный список вложен на один уровень в нумерованном списке.
